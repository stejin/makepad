## Parameters
- ``x``:  Specify the value to evaluate.
- ``y``:  Specify the value by which to perform the modulo.
## Description
`mod` returns the value of _`x`_ modulo _`y`_. This is computed as _`x`_ - _`y`_ * [[floor]](_`x`_/_`y`_).
