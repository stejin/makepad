sqrt — return the square root of the parameter
## Declaration
- ``genType sqrt(genType x)``
## Parameters
- ``x``:  Specify the value of which to take the square root.
## Description
`sqrt` returns the square root of `x`. i.e., the value
$$
\sqrt{x}
$$
Results are undefined if 𝑥<0.
